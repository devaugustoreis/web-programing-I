A tag "ul" representa "unordered list" e é utilizada para criar uma lista em que a ordem dos elementos não tem relevância. Dentro da tag "ul", precisamos usar a tag "li" (list item) para criar cada um dos itens da lista.

Podemos estilizar a lista abaixo utilizando CSS. Por padrão, uma lista não ordenada utiliza o "disc" para cada item. Contudo, podemos usar a propriedade "list-style-type" (CSS), para substituir esses "disc" por outros elementos.

A tag "ol" representa "ordered list" e é utilizada para criar uma lista em que a ordem dos elementos tem relevância. Dentro da tag "ol", precisamos usar a tag "li" (list item) para criar cada um dos itens da lista.

Por padrão, uma lista ordenada é ordenada numericamente, mas existem alguns atributos que podem ser utilizados para mudar o comportamento e ordenação da lista.

type -> Define o tipo da lista (numérica, letras, números romanos). Os seguintes valores são aceitos: 1, A, a, I, i.
start -> Define em qual elemento a lista começará. Sempre deve-se passar um valor numérico, mesmo que a lista não seja numérica.
reversed -> Define que a lista deve começar de trás para frente.

Os três tipos de listas acima tem utilizações diferentes. As tags "ul" e "ol" funcionam de modo semelhante, mas a escolha por uma ou outra está atrelada ao fato da ordem dos itens da lista ser relevante ou não. A tag "dl" (e consequentemente as tags "dt" e "dd"), embora não ofereça (por padrão) nenhum elemento visual para identificar cada item da lista, possui algumas vantagens: além da flexibilidade de aceitar imagens, links e outros, ainda possui um elemento específico (importância semântica) para a descrição dos itens da lista.
