Propriedades dos Items Flex (Filhos)

order
  • Esta propriedade controla a ordem em que cada item flex aparece em um container flex.
  • Caso não informado, o padrão para um item flex é order: 0.
  • Items com valores menores aparecerão primeiro.
  • Items com o mesmo valor aparecem na mesma ordem definida no código HTML.
  • Valores negativos podem ser atribuídos para esta propriedade.

flex-grow
  • Esta propriedade define a quantidade do espaço remanescente no container flex que cada item flex receberá.
  • O valor padrão é flex-grow: 0 e define que os items ocuparão seu espaço natural.
  • Caso algum elemento tenha flex-grow: 1 (ou mais), este elemento ocupará todo o espaço remanescente no container flex.
  • Caso vários elementos tenham flex-grow: 1 (ou mais), eles dividirão o espaço remanescente na proporção informada.
  • Exemplo: Um elemento com flex-grow: 2 receberá o dobro do espaço remanescente em relação a um elemento com flex-grow: 1.
  • Esta propriedade não aceita valores negativos.

flex-shrink
  • Esta propriedade define a capacidade que cada item flex tem de encolher-se, caso necessário, para caber dentro de um container flex.
  • O valor padrão é flex-shrink: 1 e define que o item flex pode se encolher além de sua dimensão natural para caber dentro do container flex.
  • O limite de encolhimento de um item flex é seu conteúdo interno.
  • flex-shrink: 0 define que o item flex não se encolherá, podendo extravasar os limites do container flex.
  • A quantidade que cada item flex encolhe é determinada pela proporção distribuída entre eles.
  • Esta propriedade não aceita valores negativos.

flex-basis
  • Esta propriedade define um tamanho padrão para um item flex antes que o espaço remanescente seja distribuído entre os items flex.
  • Aceita valores (px, rem, %, vw/vh) e palavras-chave: auto, content, min-content, max-content, fit-content.
  • O valor padrão é flex-basis: auto e considerará a altura e largura determinados para o elemento.
  • Flex-basis leva em consideração a direção do container flex. Para flex-direction: row, flex-basis se comportará de modo semelhante a propriedade width. Para flex-direction: column, flex-basis se comportará de modo semelhante a propriedade height.

flex
  • flex: (flex-grow) (flex-shrink) (flex-basis);
  • O valor padrao para esta propriedade é flex: 0 1 auto;
  • O segundo e o terceiro parâmetros são opcionais.
  • Se a propridade flex for usada, mas um terceiro parâmetro não for passado, então flex-basis assumirá 0% no lugar de auto.
  • É recomendado utilizar esta propriedade sintética em vez das propridades individuais (grow, shrink e basis).
